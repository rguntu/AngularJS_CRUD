* for Client(AngularJS) setup:

	* npm i -g npm@5.6.0
	* npm cache clean --force && bower cache clean
	* npm i & bower install


* for Server(NodeJS) setup:

	* Install mongoDB
		mongod --dbpath ~/mongodb/data/db
	* PORT=4000 npm start
